ここは, 私村上が執筆した[Zwegersの不定値テータ関数](https://github.com/YuyaMurakamiMath/guide_alg_num/blob/main/%E4%BB%A3%E6%95%B0%E7%9A%84%E6%95%B4%E6%95%B0%E8%AB%96%E3%81%AE%E9%81%93%E6%A1%88%E5%86%85.pdf)というpdfの保管庫です.
